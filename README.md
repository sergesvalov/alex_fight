# Alex Fight

> 3D-шутер | Godot 4.7 | Android / PC | Ретрофутуризм | P.T.-style Horror

Коридорный шутер с элементами survival horror и психологического триллера. Действие разворачивается в заброшенной провинциальной гостинице «Сибирь», которая превратилась в сюрреалистическую бесконечную петлю. Герой — полицейский без памяти (Алекс) — ищет выход на крышу, собирает голографические VHS-кассеты с отголосками своего прошлого и отбивается от жуткого проекта «Цербер».

---

## 🌟 Ключевые особенности (Последние обновления)

- **Бесконечная петля (Non-Euclidean Geometry)**: Лестничная клетка, ведущая вверх и вниз, бесшовно телепортирует игрока на тот же самый этаж, создавая пугающий эффект лабиринта в духе P.T.
- **Лазерное оружие**: Вместо классического дробовика внедрен научно-фантастический лазерный пистолет с механикой нагрева/остывания.
- **Реалистичные визуальные эффекты**: При выстрелах на стенах остаются перманентные обугленные следы от лазера со снопом искр (используется технология Decal/Multiply).
- **Компонентная архитектура**: Полностью модульный код. HUD, Искусственный Интеллект и Управление игроком разбиты на независимые компоненты, общающиеся через глобальный `EventBus`.

## 🚀 Быстрый старт

1. Открыть проект в **Godot 4.7**
2. Импортировать (`Project → Import`)
3. Запустить `scenes/levels/hotel_siberia/hotel_level.tscn`
4. Для сборки на Android: `Project → Export → Android` (также поддерживается сборка через Jenkins Docker)

> **⚠️ ОСТОРОЖНО ПРИ РАБОТЕ С АССЕТАМИ**: Проект собирается на Jenkins через Godot Headless. 
> - **Никогда** не создавайте и не коммитьте сгенерированные вручную файлы `.import` с фейковыми UID.
> - Все сгенерированные ИИ изображения должны быть строго конвертированы в формат **.jpg (JPEG)**, так как Godot Headless на CI может баговать с PNG-текстурами.

## 📁 Структура

```
alex_fight/
├── GDD.md / MECHANICS.md ← Документация
├── scenes/               ← Основные уровни (hotel_level.tscn) и префабы
├── scripts/              ← Вся логика, разбитая по модулям
│   ├── autoloads/        ← EventBus, GameStateManager, MouseManager
│   ├── player/           ← Компоненты контроллера игрока
│   ├── enemies/          ← Компоненты ИИ Цербера (Movement, Sensors, State Machine)
│   └── ui/               ← Независимые виджеты HUD (HealthBar, HeatBar)
├── entities/             ← Игровые сущности (Player, Cerberus, LaserPistol)
├── assets/               ← Текстуры и модели (только JPG!)
├── tests/                ← Автотесты (Smoke & Logic) для Jenkins
└── Jenkinsfile           ← Пайплайн сборки и тестирования
```
